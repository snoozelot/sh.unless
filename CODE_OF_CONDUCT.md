# Code of Conduct

This project is about code and technical merit, nothing else.
Stay on topic.

Something offends you?
Being offended is a choice — better choices are available, yours to make, yours to handle.
Your emotions are your own responsibility.

Having a beef with someone?
Work it out yourselves.
Nobody owes you anything, not even a reply.

We don't mandate politeness.
But you probably won't get far by insulting others — practical wisdom, not a rule.

Don't like it? Fork it!
